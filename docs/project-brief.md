# Сквозной проект: Журнал посещаемости и активности студентов

## Тема
Электронный журнал отметок посещаемости и активности на занятиях.

## Основная сущность (объект управления)
Посещение / Оценка активности (фиксация факта присутствия и оценки работы студента на конкретном занятии).

## Минимальные поля основной сущности
- Студент (ссылка)
- Занятие (ссылка)
- Статус посещения (Присутствовал / Отсутствовал / Уважительная причина)
- Оценка за активность (число / балл)
- Комментарий преподавателя

## Вторичные сущности (для расширения)
- Группа (название, курс)
- Студент (ФИО, номер зачетки, email)
- Занятие (дата, тема, тип занятия)

## Экраны (минимально)
- Главная/Сводка (Dashboard) — общая статистика: всего групп/студентов, средняя активность по группам, последние занятия.
- Список групп — перечень учебных групп.
- Список студентов — просмотр студентов выбранной группы.
- Занятия (расписание) — список созданных занятий (дата, тема).
- Форма создания занятия — создание новой даты и темы для проставления отметок.
- Журнал посещаемости — таблица "Студенты vs Занятия" для проставления отметок.
- Карточка студента (Отчет) — детальная статистика по конкретному студенту.
- Страница входа — заглушка.
